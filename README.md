# preact-indexOf

A tiny wrapper component that renders a child at the specified index.

	import IndexOf from "preact-indexOf";

    const Example = () => (
	    <IndexOf index={0}>
  	        <div>Panel 1</div>
  	        <div>Panel 2</div>
  	        <div>Panel 3</div>  	          	        
        </IndexOf>
    );
    
## Installation

	npm install preact-indexOf
